# Quest-keyboard
##何なのか
仮想空間でも快適かつ高速に日本語入力が出来る(目標)キーボードのデモアプリです。
ハンドトラッキングを前提としています。コントローラーは不要です。
対応デバイスはOculus Questのみです。
##使い方
①Assets/Scenes/QuestKeyboard.unityを開く
②ビルドする
③Oculus QuestをPCに接続する
④作成されたapkをOculus Questにインストールする
⑤Oculus Questをかぶる
⑥ハンドトラッキングモードに変更する
⑦QuestKeyboardを起動する
⑧正面の机上のキー群から黒いキーを押す
  →タイピングゲームが始まる
⑨青色の文字列と同じ文字列を入力してエンターを押す
　→親指から順に母音が設定されており、それと子音キーの組み合わせで五十音を入力します。
