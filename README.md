# Quest-keyboard
## 何なのか
仮想空間でも快適かつ高速に日本語入力が出来る(目標)キーボードのデモアプリです。
ハンドトラッキングを前提としています。コントローラーは不要です。
対応デバイスはOculus Questのみです。
## 使い方
1.  Assets/Scenes/QuestKeyboard.unityを開く
2.  ビルドする
3.  Oculus QuestをPCに接続する
4.  作成されたapkをOculus Questにインストールする
5.  Oculus Questをかぶる
6.  ハンドトラッキングモードに変更する
7.  QuestKeyboardを起動する
8.  正面の机上のキー群から黒いキーを押す
    - タイピングゲームが始まる
9.  青色の文字列と同じ文字列を入力してエンターを押す
    - 親指から順に母音が設定されており、それと子音キーの組み合わせで五十音を入力する。
